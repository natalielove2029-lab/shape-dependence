# shape dependence

This repository contains all the original codes used in the manuscript arXiv:2507.04557.

scr2d.py is a cluster counting code to obtain the sheared square corner contribution for 2d RTFIM samples
gap-stats.py is a cluster counting code to obtain the line segment corner conribution for 2d RTFIM samples
shape-dependence.ipynb is a Jupyter Notebook containing all analysis scripts including extrapolations
